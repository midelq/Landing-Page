Make sure you have **Node.js** assigned (LTS version recommended).

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/portfolio.git
   cd portfolio
   ```

2. **Install dependencies**
   ```bash
   pnpm install
   # or

   ```

   ```bash
   pnpm run build
   # or
   npm run build
   ```
